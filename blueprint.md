# Blueprint: Lotto Number Generator

## Overview

This is a simple web application that generates and displays a set of random lottery numbers. The user can click a button to get a new set of recommended numbers.

## Features

*   **Lotto Number Generation:** Generates 6 unique random numbers between 1 and 45.
*   **Display:** Shows the generated numbers on the screen.
*   **User Interface:** A clean and simple interface with a button to trigger the number generation.

## Current Plan

*   **HTML:** Create the basic structure with a title, a button, and a container to display the numbers.
*   **CSS:** Style the page for a clean and modern look, including layout, fonts, and colors.
*   **JavaScript:** Implement the logic for generating the random numbers and updating the display when the button is clicked.
